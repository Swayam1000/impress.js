Let's start with IoT
What is IoT?

The Internet of Things (IoT) is the network of physical objects or "things" embedded with electronics, software, sensors, and network connectivity which enables these objects to collect and exchange data


The IoT links smart objects to the Internet

M2M--->MACHINE TO MACHINE COMMUNICATION
What can we do with IoT?
What is BRILLO?

Brillo is an IoT operating system by Google anounced at Google I/O 2015


Brillo extends the Android platform to all connected devices, so they are easy to set up and work seamlessly with each other including smartphones.

What is WEAVE?

Protocol for synchronizing data between smart devices is called Weave.


Weave is a cross-platform common language (based on JSON) that will be available to developers.

What is Intel Galileo??

Galileo is a microcontroller board based on theIntel Quark SoC X1000,
application Processor 32-bit Intel Pentium-class system on a chip(SoC)


It is      Arduino     compatible

What is Arduino??

Arduino is an open-source computer hardware and software company, project and user community that designs and manufactures microcontroller-based kits for building digital devices and interactive objects that can sense and control the physical world.

Arduino Software Development Environment to create programs for Galileo are called SKETCHES

 
Now let's know about the Aritecture of Galileo...
Now let's take a look at Components of Galileo Board...
Board Size---> 10 X 7 (cm)
USB Connectors
Ethernet Connector
Power Jack
2 Reset Buttons(Used to reset Sketches and processor)
ReBoot Button(Used to ReBoot Processor)
Analog-Digital I/O 20 pins

#14 Digital I/O

     *6 can be used as PWM(pulse width modulation) outputs with 8/12-bits       of resolution

#6 as Analog inputs with 12 bits of resolution


#An ICSP[In-circuit serial programming] header

#A JTAG[Joint test Action Group] header 10-pins(for debugging)

STORAGE OPTIONS ON GALILEO BOARD

#PCLe(Peripheral Component Interconnect Express)

    *mini-card Slot

    *Provides USB 2.0 Host Port which supports 128 USB end point       devices


#8 MB Legacy SPI(Serial Peripheral Interface) to store firmware

#256kb/512kb dedicated SKETCH storage

#512kb embedded SRAM

#256MB DRAM

#External microSD card support upto 32gb

Prime difference between GEN1 and GEN2 Galileo Board


In GEN2 RAM is 512MB

Storage support extends to

       *Flash memory

*EEPROM

Why?
Limitations of Galileo Board

Comparitively less Speed and less RAM size

Support for only Linux based OS

IDE that can be used is only Arduino IDE

Supports only GCC and ICC compilers

No GPU

No Audio/Video Support

No Camera
